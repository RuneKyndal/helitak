Helitak HMI scraper - install

This runs as a bookmarklet, not a standalone app. Reason: the HMI's web server doesn't allow another page to read its files (CORS), so the scraper has to run inside the HMI's own page, which only works once you've actually navigated to it in Chrome.

Setup (once)

  1. Tap and hold the orange button below, then choose "Add to Bookmarks" (or copy the code in the box and create a new bookmark, pasting it as the URL).

  2. Helitak Scraper

Each time you use it

  1. Connect your phone to Wi-Fi Helitak Tank.
  2. In Chrome, go to http://192.168.1.3/ (or any specific page, e.g. the Tank 32 Landscape screen, or IO RemoteL).
  3. Open the address bar, type h or whatever shortcut text you gave the bookmark, and tap it to run it. A small dark panel appears over the page.
  4. Tap Start crawl - it follows same-origin links, downloads every page/css/js/image it can reach, and lists them live.
  5. Optionally tap Capture live data (10s) on a page that streams socket.io data, to grab a slice of raw payloads into the export too.
  6. Type any manual notes/observations into the notes box.
  7. Tap Export ZIP - downloads everything as one .zip to your phone's Downloads folder.
  8. Upload that .zip back to me here.

If "Add to Bookmarks" on long-press doesn't work on your phone, use the fallback: copy the text from the box below, create a new bookmark manually (Chrome menu → Bookmarks → add), and paste this whole thing as the URL field.
