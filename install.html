Helitak HMI scraper - install

This is a short loader bookmarklet. It doesn't carry the full scraper code (that was too long for Chrome's bookmark field) - it just fetches the real script from GitHub each time you run it, so it stays tiny and won't get truncated.

Setup (once, before you lose internet)

  1. Replace YOUR-USERNAME/YOUR-REPO below with wherever you host helitak-scraper.js on GitHub (raw.githubusercontent.com link, or GitHub Pages).
  2. Copy the whole line in the box below.
  3. Open Chrome → chrome://bookmarks → tap + to add a new bookmark.
  4. Name: HK (or anything short)
  5. URL: paste the copied text, then tap right at the very start of the field and type javascript: by hand (Chrome strips that prefix on paste, so it must be typed, not pasted).
  6. Save.

Each time you use it

  1. While still on cell data, leave the tab open or just remember this page exists - you won't need it again until next time.
  2. Connect to Wi-Fi Helitak Tank, open Chrome, navigate to the tank page you want (e.g. 192.168.1.3/Setup%2032/IO%20RemoteL).
  3. Open the address bar, type a couple letters of HK to bring up the bookmark suggestion, and tap the suggestion itself (not Enter) - or open the bookmarks list (star icon) and tap HK from there.
  4. A small dark panel appears on the page. Use Start crawl, Capture live, type notes, then Export ZIP.

Export ZIP needs internet (to fetch the JSZip library) - either do that step after switching back to cell data with the tab still open, or ask for the offline-bundled version that doesn't need this.
